# 📊 Telecom X - Análise de Evasão de Clientes (Churn)

Este projeto faz parte do **Challenge de Data Science da Alura** e tem como objetivo analisar os dados de clientes da empresa Telecom X para compreender os fatores que levam à evasão de clientes.

A evasão de clientes, conhecida como **Churn**, ocorre quando um cliente decide cancelar o serviço contratado. Identificar os fatores que contribuem para esse comportamento é essencial para melhorar estratégias de retenção.

---

# 🎯 Objetivo

O objetivo deste projeto é analisar os dados de clientes da Telecom X e identificar padrões relacionados ao cancelamento de serviços.

A análise busca responder perguntas importantes como:

- Qual a taxa de evasão de clientes?
- Quais características estão associadas ao cancelamento?
- Quais perfis de clientes apresentam maior risco de churn?
- Quais ações podem ajudar a reduzir a evasão?

# 📊 Principais Insights

Alguns padrões importantes foram identificados durante a análise:

- Aproximadamente **26% dos clientes cancelaram o serviço**.
- Clientes com **contrato mensal apresentam maior taxa de churn**.
- Clientes com **menor tempo de permanência tendem a cancelar mais**.
- Certos **métodos de pagamento estão associados a maior evasão**.

Esses resultados ajudam a identificar perfis de clientes com maior risco de cancelamento.

---

# 💡 Recomendações

Com base nos resultados da análise, algumas estratégias podem ajudar a reduzir o churn:

- incentivar contratos de longo prazo
- desenvolver estratégias de retenção para novos clientes
- incentivar métodos de pagamento automáticos
- analisar possíveis problemas de satisfação com determinados serviços
